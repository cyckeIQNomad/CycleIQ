# CycleIQ Full Site

This is a full-featured crypto blog site using Next.js and Markdown.

## 📚 Sections

- /news → Daily crypto news
- /macro → Macroeconomic updates
- /reviews → Crypto project reviews
- /courses → Course announcements

## 🚀 Setup

```bash
npm install
npm run dev
```

## ✍️ Add content

Add `.md` files to:

- `src/content/news/`
- `src/content/macro/`
- `src/content/reviews/`
